# Rolagem de Dados RPG

Um aplicativo simples em Python para rolar dados de RPG com interface gráfica.

## Funcionalidades
- Seleção de dados (d4, d6, d8, d10, d20, d100).
- Exibição do resultado da rolagem.
- Histórico das rolagens.
